## Learning goals

This project is designed to develop practical data engineering skills through constructing an end-to-end threat intelligence platform.

### Primary Learning Objective

- Develop on understanding of relational databases using PostgreSQL.
- Improve SQL proficiency, including joins, aggregations, window functions, CTEs and query optimisation.
- Build reliable ETL pipelines using Python.
- Ingest data from multiple public OSINT sources, including APIs and RSS feeds.
- Gain practical experience with Apache Spark and PySpark for large-scale data transformation.
- Design and implement data models suitable for analytical workloads.
- Implement automated data quality validation and monitoring.
- Apply software engineering best practices including Git version control, virtual environments, documentation, logging and testing.
- Learn how to translate intelligence collection requirements into scalable data engineering solutions.
- Produce analytical datasets and reports fropm processed threat intelligence data.

This project also serves as a portfolio piece demonstrating practical experience with technologies commonly used in data engineering roles.
